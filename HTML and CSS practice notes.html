HTML Notes

An HTML file always begins with <!DOCTYPE html> so that a browser knows what markup language it is reading.

Anything between the opening <html>and closing </html> tags will be interpreted as HTML code.

<> is an opening tag that must always come first and </> is a closing tag that must alwasy come last.

Titles will not appear in the document, but will appear in the tab bar.

Putting anything between <body> and </body> will insert it into ur document.

Using alt in a image code will aid the visually impaired in understanding your web page.

You can also add videos to your webpage by using <video src="myVideo.mp4" width="320" height="240" controls> Video not supported </video>. The code must include width and height to specify how large the video screen will be.

You should use <p> and </p> whenever you are writing something in the body.

<br> brand <br /> will separate two lines of writing that you put between it.

There is a whole hierarchy to HTML to help organize the code you are writing in it.

Comments can be written in HTML using the code <!-- to open it and -->.

Headings are used to emphasize words that are used to divide texts in sections.

You can emphasize words by using <em> and </em>.

You can make words bold by using <strong> and </strong>.

Putting two line breaks directly next to each other will separate two sentences.

You can make easy to read unordered lists using <ul> and </ul> and then separating the words in the list using <li> and </li>.

You can make ordered lists using <ol> and </ol> which puts numbers next to each word in the list.

You can add images to your webpage using <img src="image-location.com" />. This code goes to get the image from the source (src) that you enter in the quotes.

You can add links to an external webpage using <a href="LINK"> and then closing it with </a> but this only adds text links.

You can add an image link by wrapping the <img> tag with <a>.

You can add links to external webpages that open a new window using the target attribute.

Most webpages are made in a big folder containing smaller folders with different sections of a website. And to open each folder and see what's inside you must use <a href="./Folder Name.html">

We can make a screen go to a specific portion of the page by using

Because there is so much content, you cannot see it all on one page So you can make the page go to the top or the bottom by using <a href="#top">Top </a> to go to top and <a href="#bottom">Bottom </a>.

The tags that we have used so far are non-semantic meaning they do not describe the content inside of them. Semantic tags modify the content inside of them.

The nav element contains links to internal pages or content.

Tables can be created in HTML to display data using <table> and </table>.

To make rows in a table you must use <tr> and </tr>.

You can add data to a table using <td> and </td>.

Table data does not make sense without adding table headings using <th scope="Column or Row">The name of it </th>.

Data can span columns using the colspan attribute and uses an integer greater than or equal to 1 to denote columns. colspan="Number".

Data can span rows using rowspan and an integer greater than or equal to 1. rowspan="Number".

Long tables can be sectioned off using <tbody>.

Table heads can be sectined off using <thead>.

The bottom part of a long table can be sectioned off using <tfoot.

CSS can be used to style tables by adding borders, colours and other elements.

CSS Notes

CSS stands for Cascading Style sheets.

To style an HTML element you must open it with <style "and then add what you want to change and to what and then close it">.

To change colour and font size of text you use <p style="colour: red; font size: 20px;"> Then the text you want to change.

To change font <p style="font-family: Font Name">.

HTML allows you to write CSS code in a dedicated section of the page. It is opened by using <head> <style> "then what you want to change" { what changes you want to make }.

Most developers avoid mixing code by making separate files for HTML only code and CSS only code.

To link the CSS file to the HTML one you use <link href="File Name.css" type="text/css" rel="stylesheet">.

To change all text in a certain place you only need to put the HTML element you want to change {what you want to change about it}.

The class element is used to change certain elements of a page. We can add multiple classes by just adding different elements.

An ID attribute can be added to an element to by using the name of the element then adding a #.

Classes can be added to attributes so you can change everything with the same class easily.

There is a certain order to which elements get styled by what so you must be careful. Ex: class precedes headings.

Chaining selectors is necessary for some HTML elements and is done by creating multiple selectors.

Nested elements are used to style HTML elements inside other HTML elements.

Something even more specific than IDs are !important and they will override anything else.

It is possible to add CSS styles to CSS selectors which prevents repetitive code.

To style an HTML element using CSS, you need to write a CSS declaration inside of the body of a CSS selector.

The font refers to the font-family and that is what you need to use to change the font on a webpage.

You can also change the font's size using font-size.

Font-weight controls if the text is bold or not.

You can align the text to the right, left or center using text-align.

Changing the color can change the color of the foreground or the background (Foreground is text color).

The opacity of text can also be changed by percentage using opacity.

A background image can be put in your webpage using background-image and then putting the url.

The Box Model is how HTML positions things automatically on a webpage. Using HTML and CSS we have unknowingly learnt about aspects of the Box Model.

The 4 things that can be remodeled are the dimensions, the borders, the paddings and the margins of an element's box.

The six properties that are changeable are the height, width, padding, border, margin and overflow.

The borders can be affected by width, style and color.

The border radius modifies the corners of the box the element is inby using border-radius.

The padding of the box can be affected using padding-top, left, right, and bottom.

Using the function margin: ?px; will affect all four sides of the box. But, each specific side can be modified using margin-left, margin-right, margin-top and margin-bottom.

There is also a margin shortcut that is the same as the padding shortcut, where each margin can be changed in one large function.

Using margin: ?px auto; will set the top and bottom margins to something and then center the element.

Putting a number after an element that you want to change the margin and their sum will be the space in between.

You can set a minimum and a maximum of how large the width of a box will be using min-width and max-width.

The overflow function controls what exceeds the borders of a box. There are three changes which is to hide it, to put a scroll wheel or the default thing is to show it beneath the box.
